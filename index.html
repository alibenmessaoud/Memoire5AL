Les tests automatisés

@Maxime - Les tests unitaires - plus répandu, le plus efficace, assez simple à mettre en place - tests "boite blanche" (test à accès au code) - Les tests fonctionnels - Bon fonctionnement de l’interface graphique = Ennuyeux - Robot effectue les tâches = Si problème alors probleme dans l'interface - 2 types d'outils - test application web : Simule l'utilisation d'un navigateur web - test application native : prennent la main sur toute la machine (Inconvenient : Machine dédié)
