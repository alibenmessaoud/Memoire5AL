### Prérequis

- Installation de [Pandoc](http://johnmacfarlane.net/pandoc/)
- Installation de [LaTex](http://latex-project.org/ftp.html)

### Export en PDF

Executer la requête suivante dans le terminal :

> . generate_of_report.sh
